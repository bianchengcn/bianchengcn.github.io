Geotools对geojson的解析

在 GeoTools 中，对 GeoJSON 的支持是通过一个插件来完成的，用户同样可以在 Maven 的 pom.xml 配置文件中添加下述的依赖。

       <dependency>
            <groupId>org.geotools</groupId>
            <artifactId>gt-geojson</artifactId>
            <version>${geotools.version}</version>
        </dependency>

有关插件导进来以后，就可以根据相关的接口对Geojson进行解析

        GeometryJSON qison = new GeometryJSON() ;
        String json = "{\"type\":\"Point\",\"coordinates\":[100.1,0.1]}";
        Reader reader = new StringReader(json);
        Point p = qison.readPoint( reader ) ;
