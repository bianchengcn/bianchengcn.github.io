Outlook 启动时出现不受支持的文件夹类错误

2021/4/9

适用于:

Microsoft Outlook 2010, Microsoft Office Outlook 2007

本文内容

原始 KB 编号：   2668932

症状

请考虑以下情况：

安装适用于 Windows Vista 或 Windows 7 的 Apple iCloud 控制面板。 将 Apple iCloud 控制面板配置为启用一个或多个 iCloud 服务，然后将它们与 Microsoft Outlook 2007 或 Outlook 2010 同步。 使用 Outlook 客户端或 Windows 控制面板邮件项目将 iCloud 数据文件设置为默认文件。

在这种情况下，当您启动 Outlook 时，您将收到以下错误：

无法启动 Microsoft Outlook。 无法打开 Outlook 窗口。 无法打开文件夹集。 IPF 中不支持 (类。联系人)

原因

Apple iCloud 数据文件使用自定义数据文件 (.aplzod) 结构。 Outlook 无法将数据文件用作默认数据文件。

解决方案

若要将 Outlook 数据文件 (.pst) 设置为默认值，请按照以下步骤操作：

选择“开始”，然后选择“运行”。

复制并粘贴，或在"打开"框中 键入控制面板，然后按 Enter：

备注

如果使用的是 64 位版本的 Windows，请双击"控制面板"中的"查看 32 位控制面板"以显示" 邮件" 图标。

打开 "邮件设置" 对话框：

对于 Windows 7：

如果"控制面板"位于"大图标"或"小图标"视图中，请选择"邮件 "。 将显示 "邮件 设置"对话框。

If Control Panel is in Category view， under View by， select either Large icons or Small icons， and then select Mail. 将显示 "邮件 设置"对话框。

对于 Windows Vista 或 Windows XP：

如果"控制面板"位于经典视图中，请双击"邮件 "。 将显示 "邮件 设置"对话框。

If Control Panel is in Category View， under Control Panel， select Switch to Classic View for Windows XP， or select Classic View for Windows Vista， and then double-click Mail. 将显示 "邮件 设置"对话框。

在"邮件设置" 对话框中，选择"显示配置文件"。

选择当前 Outlook 配置文件，然后选择"属性 "。

在"邮件设置 - 配置文件 名称"对话框中，选择 "数据文件"。

在" 帐户设置" 对话框中，选择" 数据文件" 选项卡。

选择 Outlook 数据文件。 Outlook 数据文件具有 .pst 扩展名。

选择" 设置为默认值" 按钮。

选择“关闭”。

启动 Outlook。
