在vue中使用MD5

第一步安装MD5

npm install --save js-md5

 

第二部引入全局

import md5 from 'js-md5';

Vue.prototype.$md5 = md5;

2.2、全局使用

将您需要加密的信息放进去：

this.$md5('This is encrypted content')  // 6f43dd5db792acb25d6fe32f3dddac70

3、局部引入

3.1、首先安装依赖项，然后引入

import md5 from 'js-md5'; 

3.2、局部使用

md5('This is encrypted content')  // 6f43dd5db792acb25d6fe32f3dddac70
