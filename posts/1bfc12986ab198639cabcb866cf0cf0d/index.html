安装西门子博图一直重启_博图软件常见错误解决方法

1、安装SIMATIC STEP7 Professional一直停留在修改系统估计剩余时间：2分钟。

解决方法：

在个人计算机“控制面板”中点击“程序和功能”，点击左侧“打开或关闭WINDOWS功能”，将第六项MSMQ这一项选中。

2、修改系统和组态系统没有正常完成。(安装时关闭了杀毒软件和防火墙)安装错误提示如下

解决方式：

确定是否系统兼容，如果不兼容，建议重装系统。

系统运行时间长的话，可能也会出现类似问题，所以安装西门子软件最好全新 做一遍系统，然后安装一个office，先别装杀毒软件。

3、博图软件安装提示无限重启

这个问题基本上所有西门子软件安装都会出现，需要删除注册表信息。

点击“开始-右键-运行”，输入“regedit”，HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager，删除“PendingFileRenameOperations”，删除后不要重启电脑了，直接安装。

4、安装过程中提示部分文件已损坏

此问题由于下载或解压过程出现中断，或者电脑中安装杀毒软件和360等造成博途软件误删导致。

解决方法：卸载杀毒软件，重新下载博途安装文件，重新安装。

5、安装过程中提示“Command line option syntax error type command /?for help ”

这个问题主要是系统的原因导致的或安装路径中存在中文。

解决方法：可以先忽略继续安装，正常情况下不会对使用造成影响。如果之后出现报错，就需要重装系统。如果是路径原因，需要重新解压安装。

6、提示应用程序并行配置不正确

a.开始-运行输入“services.msc”，确定，打开服务(本地)；在窗口中找到“Windows Modules Installer”服务，查看是否被禁用；若禁用，需改为启用-手动，重启计算机后安装应用程序。

b.若果以上方式无作用，那可能是系统缺少“Microsoft Visual C++20XX(运行库)”文件。搜素下载并安装。

7、博途软件安装提示需要安装Net 3.5 SP1

解决方法：控制面板-程序-启用或关闭Windows功能-勾选NET 3.5

8、安装完成，打开运行软件时，提示找不到许可证“step7 professional或者basic”

这是因为没有授权或者授权管理器工作不正常，或者系统不兼容。

解决方法：打开授权管理器，查看是否授权。

打开服务管理器，查看almservice服务有没有运行。

授权文件一般是隐藏文件，查看方法：进入系统盘C盘，在菜单栏找到“工具”，查看文件夹选项去掉隐藏选项，就能找到C：/AX NF ZZ。

授权有时候会不成功，多试几次，同时要注意授权文件版本问题。我安装的V14版本博途软件，也是尝试了好多次才成功的。

注如果授权不成功，无法添加设备。
