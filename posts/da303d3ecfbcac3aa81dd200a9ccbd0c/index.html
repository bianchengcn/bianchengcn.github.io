VB语言通用基础语句

一．赋值语句

变量的赋值
格式：
［Let］ <变量> ＝ <表达式>
功能：先计算赋值号右边表达式的值，然后将值赋值给变量。
注意：
1.“=” 意义是“赋值”，x = y和y = x不等价，前者是将y的值赋给x，后者是将x的值赋给y；
2.“=”左边只能是变量，不能是数值或表达式；
3.一次只能赋值一个
例如：
sum = 0 
 i = 0
不能写成：
sum = i = 0

对象属性的值设置
格式：对象名.属性名 = 属性值
R = Val（Text1.Text）
例如：
dim a 
Command1.Caption = “天才”
a=123456
Text2.Text=str（a）

二.打印语句

对象名.Print 【指定对象名1，指定对象名2】
功能：在指定的对象上输出指定元素的值。
例如：
Private Sub Command1_Click（）
a = 123
b = 456
c = 789
Form1.Print a; b; c ‘输出在窗体Form1上
Print “霍华德”， “布克”， a + b + c ‘输出在窗体Form1上
Picture1.Print “霍华德”， “布克”， a + b + c ’输出在图片框Picture1上
Debug.Print “霍华德”， “布克”， a + b + c ‘输出在立即窗口上
Printer.Print “霍华德”， “布克”， a + b + c ’在指定打印机上输出
end sub 
注意：
1.print只能在窗体、图片框、打印机、立即窗口上输出内容
2.若要在窗体上实现输出时应在窗体的属性面板设置AutoRedraw属性=True，若不设置或将无法显示

Print的相关函数
1.Tab（）函数
Tab（A）
功能：把光标移到由A确定的位置。
2.Spc（）函数
Spc（A）
功能：光标移到A个空格之后的位置。
3.Cls方法
［<对象名>］. Cls
功能：清除由Print方法显示的信息。
4.Format$函数
Format$（表达式，格式字符串）
功能：将数值、字符串或日期等数据按照指定的格式生成字符串

三.注释语句

Vb中添加注释语句的方法有多种：
1.在字符“’”（即单引号）后加注释语句或内容。
2.也可以采用Rem 注释内容的形式来注释语句或内容。
3.菜单栏右击，选择“编辑”选项，出现的编辑选项卡中从右往左数第五个是取消注释，从右往左数第六个是加注释，非常实用

四.结束语句

语法格式：end
功能：强制程序结束运行
