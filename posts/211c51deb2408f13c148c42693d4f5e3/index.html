doris建表报错:Failed to find 3 backends for policy

具体报错：

mysql> CREATE TABLE `aaa` (
    ->   ORDER_PROCESS_WT_OUT int(10)
    -> ) ENGINE=OLAP
    -> DUPLICATE KEY(`ORDER_PROCESS_WT_OUT`)
    -> DISTRIBUTED BY HASH(`ORDER_PROCESS_WT_OUT`) BUCKETS 64
    -> PROPERTIES (
    -> "replication_allocation" = "tag.location.default:2",
    -> "in_memory" = "false",
    -> "storage_format" = "V2",
    -> "storage_medium" = "HDD"
    -> );
ERROR 1105 (HY000): errCode = 2, detailMessage = Failed to find 2 backends for policy: cluster|query|load|schedule|tags|medium: default_cluster|false|false|true|[{"location" : "default"}]|HDD

解决办法：

添加：

"storage_medium" = "SSD"

完整SQL：

CREATE TABLE `aaa` (
  ORDER_PROCESS_WT_OUT int(10)
) ENGINE=OLAP
DUPLICATE KEY(`ORDER_PROCESS_WT_OUT`)
DISTRIBUTED BY HASH(`ORDER_PROCESS_WT_OUT`) BUCKETS 64
PROPERTIES (
"replication_allocation" = "tag.location.default:3",
"in_memory" = "false",
"storage_format" = "V2",
"storage_medium" = "SSD"
);
