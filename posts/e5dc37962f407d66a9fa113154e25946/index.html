vue中使用MD5加密和base64加密

MD5：

安装：npm install --save js-md5

在main.js引入
import md5 from 'js-md5';

Vue.prototype.$md5 = md5;


使用加密：
"password": this.$md5(this.loginFrom.pwd)

base64：

安装：npm install --save js-base64


当前页面直接引入:
let Base64 = require('js-base64').Base64;

//使用
password: Base64.encode(this.loginForm.password)
