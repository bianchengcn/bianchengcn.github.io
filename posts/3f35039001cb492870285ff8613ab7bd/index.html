centos7.x安装docker-ce

环境：

  • 系统：centos7.x
  • docker版本：19.03.2
  • 安装方式：yum
  • 参考官方安装文档：https://docs.docker.com/install/linux/docker-ce/centos/
  • 官方命令行使用文档：https://docs.docker.com/engine/reference/run/
  • docker三剑客的区别：https://www.zhihu.com/question/277356093

安装

1.安装所需的软件包

sudo yum install -y yum-utils \
  device-mapper-persistent-data \
  lvm2

2.安装yum源

sudo yum-config-manager \
    --add-repo \
    https://download.docker.com/linux/centos/docker-ce.repo

3.安装docker-ce
sudo yum install docker-ce docker-ce-cli containerd.io

4.启动docker
sudo systemctl start docker

5.测试docker是否正常运行
sudo docker run hello-world
