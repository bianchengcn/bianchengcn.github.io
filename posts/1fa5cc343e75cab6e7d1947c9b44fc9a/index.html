Typora导出word

Typora是一款简洁易用的Markdown编辑器,

Pandoc是一个文档转换工具，可以将Markdown格式的文档转换为其他格式，如HTML、PDF等.

linux下安装 Pandoc :

sudo apt install -y pandoc

安装成功后,typora 会自动监测到.

然后 点击文件->导出->Word(.docx)即可.
