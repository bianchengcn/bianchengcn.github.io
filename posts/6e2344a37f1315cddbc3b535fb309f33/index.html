git clone 使用用户名和密码

git clone http://邮箱(或用户名):密码@仓库

示例:

邮箱 xw@qq.com
密码: xw
仓库: http://git.test.com/abc/demo
注意: 邮箱中的 @ 要使用 %40 代替。仓库 URL 不需要 http:// 。

命令:

git clone http://xw%40qq.com:xw@git.test.com/abc/demo
