一、准备工作

链接工具：Xshell （作者使用Xshell7）
Linux系统：CentOS（作者使用CentOS7.9）

二、Linux系统宝塔安装

常用Linux系统安装命令
Centos安装脚本

yum install -y wget && wget -O install.sh https://download.bt.cn/install/install_6.0.sh && sh install.sh ed8484bec

Ubuntu/Deepin安装脚本

wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh ed8484bec

Debian安装脚本

wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && bash install.sh ed8484bec

万能安装脚本

if [ -f /usr/bin/curl ];then curl -sSO https://download.bt.cn/install/install_panel.sh;else wget -O install_panel.sh https://download.bt.cn/install/install_panel.sh;fi;bash install_panel.sh ed8484bec

国产龙芯架构安装脚本（其他CPU请勿使用）

wget -O install_panel.sh https://download.bt.cn/install/0/loongarch64/loongarch64_install_panel.sh && bash install_panel.sh ed8484bec

注意：
1、服务器宝塔内外网地址、账号、密码千万不要泄露

宝塔使用方法

1、输入命令：bt

2、选择您所需要的宝塔执行操作（按提示输入编号），例如查看面板信息输入14

3、通过浏览器进入面板复制出面板地址直接访问 (根据自己的ip进行访问)

4、登录面板 （根据自己的面板信息账号密码进行登录）如密码不清晰输入bt 再输入5 进行修改面板密码
5、登录成功 （登录成功如下图）

7、面板设置

三、Linux系统宝塔卸载面板

1、输入命令：wget http://download.bt.cn/install/bt-uninstall.sh

2、接下来再输入执行脚本命令：sh bt-uninstall.sh

此时出现按提示操作选择1或者2 如输入1

卸载完成 校验 输入命令查看：bt
