vim 跳转到指定行

常用的三种方式

第一种：

vim +n file
eg:
vim +3 test.py    查看test.py文件，光标定位到第三行

第二种：
vim 进入查看文件，命令行下输入

:n          回车以后就可以见到光标跳转到第 n 行

第三种：
vim 进入查看文件，esc模式下输入

ngg   或者
nG
n 是行号，输完以后可以马上看到光标跳转
另外，如果你想查看行号

参见 查看行号
