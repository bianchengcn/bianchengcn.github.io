qt的main函数（程序启动入口）

函数入口的参数

这就是Qt中最简单的一个main函数：

int main(int argc, char *argv[])
{
    QApplication a(argc, argv);
    Widget w;
    w.show();
    return a.exec();
}

其中int argc, char *argv[]参数是很有用的。

使用.\release\程序名.exe 模型名.model 模型文件所在的地址  这种方式是在商业软件开发过程中常用的方式。最大的一个好处就是可以用脚本来启动qt程序。

关于声明对象：

经常可以看到这样的代码：

 Widget w;
 w.show();

这样的代码其实表明了不用去new一个新对象，就用栈上的那种声明方法就行，不用管声明或调用完之后的各种释放操作。
