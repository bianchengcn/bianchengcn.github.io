Markdown是什么意思

● markdown 指的是一种文本格式, 是一种轻量级标记语言, 大多都是程序员在使用, 不管什么语言的程序员, 其实都比较喜欢这种格式.

● 他是一种极简的文本格式, 用最简单的方式写出带有格式的内容

● 对于程序员来说, 写出来的文本类内容, 是不是好看并不那么重要, 我们主要关注内容.

○ 也就是说, 只要标题清晰, 有个代码段能让我比较好的格式的写一些代码

○ 大致的缩进能区别出标题级别也就可以了

○ 多了就是一个 图片, 超链接, 表格啥的, 如果能有就最好了

● markdown 就是这样一种格式的文本

如何查看 markdown

● markdown 格式文本, 文件后缀是 .md

● 只要打开编辑器基本上都能编辑

● 但是如果你想单独查看 markdown 格式文档

● 那么需要下载一个可视化软件

○ 比如你要看 word 文档, 需要安装一个 office

○ 比如你要看 mp4 文件, 需要安装是一个 快播

○ 是一个道理

○ 那么你要看 markdown 就要下载一个能适配的软件

● 一般来说, 我们会使用 typora 软件来查看

○ 因为这个软件比较轻量, 也很方便

markdown 的常用语法

1.  标题

● 在 markdown 内如果需要书写标题

● 直接书写 # 就可以了

● 一个 # 对应我们的 H1 标签

● 两个 ## 对应我们的 H2 标签

● 以此类推

2.  列表

● 如果需要无序列表直接书写 中划线(-) 加上空格就可以了

● 如果需要有序列表直接书写 数字 加 点 加上空格就可以了

3.  超链接

● 直接书写 []()

○ 在 [] 内书写文本内容

○ 在 () 内书写连接地址就好

4.  图片

● 直接书写 ![]()

○ 在 () 内书写图片地址就好了

5.  引用书写

● 直接书写 > 加空格就好了

6.  代码段书写

● 书写三个反引号 (```) 加上语言种类就好了

○ 例如: 我们要写 js, 那么就写 ```js

○ 在代码段结尾处在写三个 反引号(```)

7.  分隔符

● 这个就非常简单了

○ 写三个中划线 (---) 或者三个星号 (***) 都可以

8.  文本强调

● 加粗 : 用 ** 包裹

● 斜体 : 用 * 包裹

● 删除线 : 用 ~~ 包裹

9.  表格

○ 相对比较复杂, 但是核心就是 竖线( | )

○ 还有一个就是对其方式

■ :--- 左对齐

■ :---: 居中对齐

■ ---: 右对齐

最后

● markdown 一般我们是不考虑颜色问题的

● 如果你偏要给你的 markdown 添加一些文本颜色或者其他样式之类的

● 那么我们就可以完全依靠 html + css 的方式来添加, 但是这个方式比较鸡肋
