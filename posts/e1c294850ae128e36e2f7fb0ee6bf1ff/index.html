spring boot linux部署

linux强制杀死某个端口进程命令，例如80端口：

sudo fuser -k -n tcp 80

 linux启动jar包，配合静态资源

 java -cp static -jar ***.jar

 linux运行命令

//&代表在后台运行。
//特定：当前ssh窗口不被锁定，但是当窗口关闭时，程序中止运行。
java -jar XXX.jar &

//nohup 意思是不挂断运行命令,当账户退出或终端关闭时,程序仍然运行
nohup java -jar XXX.jar &
