IDEA关闭git提交代码时的代码检测

方案一

主要有三个步骤:

  1. 打开settings
  2. 找到Version Control
  3. Commit Dialog

方案二

默认情况下的git提交会出现执行代码分析的提示框。

提交之前会弹出下面这个框。

把"Perform code analysis" 和 “Check TODO” 复选框前面的勾去掉就好了.
