2008服务器系统关闭,Windows server 2008系统默认服务哪些可以关闭?

服务器系统在安装好之后，有很多服务都是默认开启的，但有些服务我们平常用不到，哪些服务是可以关闭的呢?

1. Computer Browser(默认禁用) :这个服务等同于网络和共享中心的网络发现，有了网络发现此应用可放心关闭。

2. Distributed Link Tracking Client :个人用户一般用不到，设置为手动。

3. Distributed Transaction Coordinator :个人用户平常用的不多，也容易受到远程拒绝服务攻击，建议设置为手动。

4. Remote Registry :这个服务应该算是必停的服务之一，设置为禁用。

5. Secondary Logon ：对于多用户，可以给某用户分配临时的管理员权限，一般不会用到，设置为手动。

6. TCP/IP NetBIOS Helper ：如果你的网络不使用NetBIOS或是WINS，设置为手动。

7. Terminal Services ：用于远程桌面控制，存在安全隐患，设置为手动。

8. Windows Defender ：它会拖慢你机器，当然前提是你不需要这个功能，设置为禁用。

9 Windows Error Reporting Service ：生成错误报告，设置为手动。

以上这些就是小编总结的windows2008可关闭的一些服务，可以根据使用情况来选择关闭不需要的一些服务。
