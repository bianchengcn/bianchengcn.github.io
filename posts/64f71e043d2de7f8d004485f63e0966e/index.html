【MicroPython】Picoweb加载静态资源错误

最近用ESP32做WEB界面，使用Picoweb做WEB服务，但是需要加载静态资源时出现错误
Traceback (most recent call last):
File “/lib/picoweb/init.py”, line 206, in _handle
File “/lib/picoweb/init.py”, line 291, in handle_static
File “/lib/picoweb/init.py”, line 276, in sendfile
File “/lib/pkg_resources.py”, line 16, in resource_stream
AttributeError: ‘module’ object has no attribute 'path’
几乎翻遍了百度没发现解决方法，看来这个组件很少有人用，只好到bing翻翻，终于在GITHUB上找到解决方案，特此记录一下，免得下次找不到了。

其实解决的方法也很简单，只要把

app = picoweb.WebApp(__name__)

改为

app = picoweb.WebApp(None)

就行了。
