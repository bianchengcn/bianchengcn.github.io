题目

请实现一个函数，把字符串 s 中的每个空格替换成"%20"。

示例 1：

输入：s = "We are happy."
输出："We%20are%20happy."

限制：
0 <= s 的长度 <= 10000

代码

Python
# 思路：
# 字符串不可变，故新建list存储新字符串
# 复杂度：
# O(N)
class Solution:
    def replaceSpace(self, s: str) -> str:
        res = []
        for c in s:
            if c==" ": res.append("%20")
            else: res.append(c)
        return "".join(res)
C++
class Solution {
public:
    string replaceSpace(string s) {
        string res;
        for (auto c:s) {
            if (c==' ')
                res += "%20";
            else 
                res += c;
        }
        return res;
    }
};
