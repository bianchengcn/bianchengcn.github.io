10.14 给定一个按非递减顺序排序的整数数组 arr，返回每个数字的平方组成的新数组，要求也按非递减顺序排序L

注意JS来里面数组的sort（），直接排序是按照字符串进行排序的，对数字：arr.sort(function(a, b){return a - b});

var sortedSquares = function(arr) {
    var len = arr.length;
        var arr1 = [];
        for (var i = 0; i < len; i++){
            arr1[i] = arr[i] * arr[i];
        }
        arr1.sort(function(a, b){return a - b});
        return arr1;
};
