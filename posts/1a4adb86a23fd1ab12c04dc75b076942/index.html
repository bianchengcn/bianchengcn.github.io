Linux运行环境搭建系列-JDK安装

JDK安装

构建环境：腾讯云CentOS 7.9。

  1. JDK jar包下载地址：https://www.oracle.com/java/technologies/downloads

  2. 使用scp命令上传Jar包到/usr/local目录下：

    $ scp jdk-8u152-linux-x64.tar.gz root@81.70.247.41:/usr/local

  3. 解压缩

    $ cd /usr/local && tar zxvf jdk-8u152-linux-x64.tar.gz && rm -f jdk-8u152-linux-x64.tar.gz

  4. 设置环境变量

    $ yum install vim && vim /etc/profile
export JAVA_HOME=/usr/local/jdk1.8.0_152
export PATH=$PATH:$JAVA_HOME/bin

  5. 退出vim编辑器，使环境变量设置立即生效

    $ source /etc/profile

  6. 查看JDK版本

    $ java -version
