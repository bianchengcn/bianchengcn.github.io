html中禁止网页另存为、查看源文件、屏蔽鼠标右键

在HTML网页中加入几行简单的代码可以禁止他人使用鼠标右键和“另存为”命令。 

一、屏蔽鼠标右键 
把网页的标签修改为下面的代码：

＜body oncontextmenu=self.event.returnValue=false onselectstart="return false"＞ </body>

现在点击鼠标右键就不会有任何反应了。 

二、禁止“另存为”命令 
在目标网页末尾“</BODY></HTML>”的标签前面加上加入如下代码，可以使“另存为”命令不能顺利执行。 

<noscript> 
<iframe scr="*.htm"></iframe＞ 
</noscript>

加入上述代码后，当执行“另存为”命令时，会弹出“保存网页时出错”的对话框。 

三、不用js也可以控制禁止右键!方法: 
<body οncοntextmenu="return false"></body> 

<!-- 禁用右键: --> 

<script> 
    function stop(){ 
        return false; 
    } 
    document.oncontextmenu=stop; 
</script> 

<body onselectstart="return false"> 取消选取、防止复制 

οncοpy="return false;" oncut="return false;" 防止复制

    function clear(){ 
        Source=document.body.firstChild.data;
