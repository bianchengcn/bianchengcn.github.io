MySQL 报错：Parameter ‘@XXX‘ must be defined.

报错信息：

Parameter ‘@Name’ must be defined.

解决方案：

连接语句中添加：allowuservariables=True; 即可

配置解析：

AllowUserVariables：是否允许 SQL 中出现用户变量，默认 false
