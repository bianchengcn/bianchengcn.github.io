【LeetCode】从尾到头反过来返回每个节点的值（用数组返回）day03

题目

输入一个链表的头节点，从尾到头反过来返回每个节点的值（用数组返回）。

示例 1：
输入：head = [1,3,2]
输出：[2,3,1]
限制：
0 <= 链表长度 <= 10000

        Stack<ListNode> stack =  new Stack<ListNode>();

第三种解决方案

//纯O（n）的复杂度
class Solution {
    public int[] reversePrint(ListNode head) {
        ListNode temp = head;
        int count = 0;
        while(temp != null ){
            temp = temp.next;
            count++;
        }
        int[] resultV2 = new int[count];
        temp = head;
        for(int i = count-1; i>=0; i--){
            resultV2[i] = temp.val;
            temp = temp.next;
        }
            return resultV2;
    }
}
