微信小程序：map地图自动缩放自适应大小

文档：

  • https://developers.weixin.qq.com/miniprogram/dev/api/media/map/wx.createMapContext.html
  • https://developers.weixin.qq.com/miniprogram/dev/api/media/map/MapContext.includePoints.html

map提供了 scale 参数，用于大小缩放，不过在需要标记多个点的时候，更期待的是根据地图上显示的内容，自动缩放地图到合适的比例。

<map id="map" show-location></map>

实现代码

// 获取地图实例
let MapContext = wx.createMapContext('map');

// 换成具体的坐标点
points = [{longitude, latitude}];
MapContext.includePoints({
  points: points,
  // 缩放视野展示所有经纬度 [上,右,下,左]  
  padding: [140, 60, 260, 60],
});
