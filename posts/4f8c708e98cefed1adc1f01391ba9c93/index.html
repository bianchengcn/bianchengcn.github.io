C语言--函数指针的使用

定义：函数指针，顾名思义就是指向函数的指针。

使用场合：

1. 需要在同一个函数接口使用不同函数。如，对不同驱动初始化函数的切换。

2. 注册回调函数，由用户定义函数执行内容，系统自动执行。

...

示例代码：

1. 在结构体中定义及其使用：

/* 代码可直接在VS中运行 */
#include <stdio.h>
#include <stdint.h>
#include <stdlib.h>

typedef struct {
    int a;
    int (*func)(int a, int b);    //定义函数指针
} ts_t;

int add(int a, int b)
{
    return a + b;
}

int main()
{
    ts_t ts;
    ts.func = add;              //传递add的函数指针

    int ret = ts.func(1, 2);    //使用
    printf("ret = %d \n", ret);
    
    system("pause");
    return 0;
}

运行结果：ret = 3

2. 定义函数指针类型及其使用

/* 代码可直接在VS中运行 */
#include <stdio.h>
#include <stdint.h>
#include <stdlib.h>

typedef int (*func)(int a, int b);    //定义函数指针类型

int add(int a, int b)
{
    return a + b;
}

int main()
{
    func myFunc = add;                //传递add的函数指针

    int ret = myFunc(1, 2);           //使用
    printf("ret = %d \n", ret);

    system("pause");
    return 0;
}

运行结果：ret = 3
