JavaScript【FS】文件系统模块

JavaScript文件系统模块（fs）

用于文件的操作

文件分为目录形式和文件形式

例如：写一个功能要求把数据库的数据导出来

再者：老板说我现在有50w条数据，要把这50w条数据写入数据库

 实践：创建文件夹（mian)添加fs.js其中fs.js写入下面的代码，运行查看结果

//引入文件系统模块（fs）
const fs = require('fs');
//查看文件的状态：目录形式，文件大小
var a = fs.statSync('../mian');
console.log(a);
//是否为目录
console.log(a.isDirectory());
//是否为文件
console.log(a.isFile());
console.log('结束')

翻译：

使用这个模块必须引入模块

1，查看文件的状态

        statSync（文件的路径）/stat(文件路径,回调函数）

                isDirectory ()      是否为目录

                isFile()                是否为文件

文件管理函数属于异步调用函数

在执行异步方法stat的时候，会从线程池中获取一个线程执行这个任务，执行完以后的结果是以回调函数放入事件队列；等主程序执行完再执行事件队列的时候，就可以获得结果；同时会把线程放回线程池

 2，创建目录

       

 mkdir(目录的路径)/mkdir(目录的路径，回调函数)

示例：创建一个名字叫1的文件夹

 //异步
//引入文件系统模块（fs）
const fs = require('fs');
fs.mkdir('./1', function(err) {
    //第一个参数，err可能出现的错误
    //第二个参数，只有成功的结果书记才会有
    if (err) {
        throw err
    }
    console.log('ok!')
})
//同步
fs.mkdirSync('./1')

3，移除目录

rmdirSync(目录的路径）/rmdir(目录的路径，回调函数）

示范：移除上面常见的文件

fs.rmdirSync('./1');
//只能移除空目录
