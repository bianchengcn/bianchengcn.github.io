Dependency introduction

Dependency Walker 是 Microsoft Visual C++ 中提供的非常有用的 PE 模块依赖性分析工具。主要功能如下：

　　查看 PE 模块的导入模块。

　　查看 PE 模块的导入和导出函数。

　　动态剖析 PE 模块的模块依赖性。

　　解析 C++ 函数名称。

　　Dependency Walker的另一个重要用途就是分析DLL(动态链接库)的详细信息，适合有经验用户进行分析。
