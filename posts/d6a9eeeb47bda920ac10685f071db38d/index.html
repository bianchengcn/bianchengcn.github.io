java学习笔记之使用MyEclipse写一个简单的登陆界面（二）

之前那个是第一种方法。还有一种方法可以直接在网页之间进行跳转。不需要通过servlet。

1、编写登陆界面。

在WebRoot下新建一个文件夹。demo02。在这个文件夹中编写login.jsp 与 hello.jsp。

2、login.jsp的编写。基本上与前面一种方法相同。只是action那里有所不同。

<%@ page language="java" import="java.util.*" pageEncoding="UTF-8"%>

<html>
  <head>
    <title>登陆界面</title>
  </head>
  
  <body>
   <center>
  	<h1>信息系统</h1>
  	<form action="hello.jsp" method="post">
	    用户名：<input type="text" name = "name"> <br>
	  密 码：<input type="password" name = "psw"><br>
	     <input type="submit" value="登陆">
 	 </form>
  </center>
  
 
  </body>
</html>

3、hello.jsp的编写。 

<%@ page language="java"     pageEncoding="UTF-8"%>
<html>
<head>
<title>用户界面</title>
</head>
<body>
	<%
	String name = request.getParameter("name");
	String psw = request.getParameter("psw");
	if(name.equals("admin")&&psw.equals("123")){
		out.println("登陆成功");
	}else{
		request.getRequestDispatcher("login.jsp").forward(request, response);
	}
%>
	
</body>
</html>
4、启动服务器，运行。结果和第一种方法一样。
