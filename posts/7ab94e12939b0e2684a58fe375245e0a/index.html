ubuntu安装nvidia显卡驱动、cuda-toolkit、cudnn

使用显卡进行深度学习需要分别安装3个驱动包：nvidia显卡驱动、cuda-toolkit工具包、cudnn工具包，需要根据显卡型号和各版本差异选择对应版本安装。
具体显卡驱动版本和cuda-toolkit版本对照查看官方文档，外网访问可能会比较慢。

# 命令行安装显卡驱动和cuda都会有问题，已踩坑，以下命令仅作参考：
# sudo apt-cache search package nvidia-driver
# sudo apt install nvidia-driver-515-server # 显卡驱动
# sudo apt-cache search package nvidia-cuda
# sudo apt install nvidia-cuda-toolkit # cuda

官网提供的cuda-toolkit安装包内置了对应版本的显卡驱动，只需要下载cuda包默认安装即可同时安装显卡驱动，即只需查看下载安装与你的显卡型号匹配cuda安装包 即可。
如果想下载安装包再安装可查看驱动下载链接、cuda-toolkit下载链接、cudnn下载链接。
安驱动需要用到gcc和make，可以用 sudo apt install gcc make 下载安装。
安装显卡驱动时若遇到报错“The Nouveau kernel driver is currently in use by your system. This driver is…”则需要禁用Nouveau驱动。禁用Nouveau驱动后进不了系统 可以查看解决方案。
我的系统是Ubuntu20.04 server，显卡版本Rtx2060，于是选用：

# 下载cuda安装包
wget https://developer.download.nvidia.com/compute/cuda/11.8.0/local_installers/cuda_11.8.0_520.61.05_linux.run
sudo sh cuda_11.8.0_520.61.05_linux.run
# or
wget https://developer.download.nvidia.com/compute/cuda/11.7.1/local_installers/cuda_11.7.1_515.65.01_linux.run
sudo sh cuda_11.7.1_515.65.01_linux.run
# or
wget https://developer.download.nvidia.com/compute/cuda/11.7.0/local_installers/cuda_11.7.0_515.43.04_linux.run
sudo sh cuda_11.7.0_515.43.04_linux.run
# or
wget https://developer.download.nvidia.com/compute/cuda/11.6.2/local_installers/cuda_11.6.2_510.47.03_linux.run
sudo sh cuda_11.6.2_510.47.03_linux.run
# 更多版本自行查看官网

cuda-toolkit装好后需要配置~/.bashrc文件，末尾添加：

# cuda的实际版本取决于你的安装版本
export PATH=/usr/local/cuda-11.8/bin${PATH:+:${PATH}}
export LD_LIBRARY_PATH=/usr/local/cuda-11.8/lib64${LD_LIBRARY_PATH:+:${LD_LIBRARY_PATH}}

然后更新配置：

 source ~/.bashrc

避坑：下载cudnn时，我之前选择的是deb包，然鹅这样安装不报错但是没啥效果，在网上找了半天，最后选择“Local Installer for Linux x86_64 (Tar)”下载解压，具体参考别人的另一篇博客。
将解压后的库文件拷贝到cuda的安装目录：

# cuda的实际版本取决于你的安装版本
sudo cp include/cudnn.h /usr/local/cuda-11.*/include/
sudo cp lib64/libcudnn* /usr/local/cuda-11.*/lib64/
sudo chmod a+r /usr/local/cuda-11.*/include/cudnn.h
sudo chmod a+r /usr/local/cuda-11.*/lib64/libcudnn*
