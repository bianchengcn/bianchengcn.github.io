SQL 常见函数整理 _ DATEPART() 从日期时间表达式中提取指定的日期部分

1. 用法
用于从日期时间表达式中提取指定的日期部分。

2. 语法

DATEPART (datepart, date)

3. 应用示例

--示例
SELECT DATEPART(datepart,'2020-5-20 13:14:32.1234567 +05:10')

注意：
① 当DATEPART ( weekday, date ) 的值等于1或者7时，是周末。

--返回值： 1 
SELECT DATEPART(weekday,'2023-10-08 13:14:32.1234567 +05:10')
