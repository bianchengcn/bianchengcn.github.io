Linux下CUDA版本切换

cp@ubuntu:/usr/local$ ls
bin  cuda  cuda-10.0  cuda-10.1  etc  games  include  lib  man  sbin  share  src
cp@ubuntu:/usr/local$ sudo rm -rf cuda
[sudo] cp 的密码： 
cp@ubuntu:/usr/local$ ls
bin  cuda-10.0  cuda-10.1  etc  games  include  lib  man  sbin  share  src
cp@ubuntu:/usr/local$ sudo ln -s /usr/local/cuda-10.0 /usr/local/cuda
cp@ubuntu:/usr/local$ ls
bin  cuda  cuda-10.0  cuda-10.1  etc  games  include  lib  man  sbin  share  src
