css input placeholder属性 样式修改 包括颜色 大小 位置

<input type="text" placeholder="请输入手机号">

CSS

input::-webkit-input-placeholder {
       /* placeholder颜色  */
	    color: #aab2bd;
	/* placeholder字体大小  */
	    font-size: 12px;
	/* placeholder位置  */
	     text-align: right;
	 }
	 input {
	         border: 1px solid red;
	  }
