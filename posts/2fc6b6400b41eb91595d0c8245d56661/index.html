C 数组逆序输出

编写程序对固定内容的数组进行逆序输出，第一个值和最后一个值的位置互换。

效果

完整代码

#include<stdio.h>
#define N 10
int main()
{
    int a[N]={10,100,20,43,54,15,6,77,82,91};
    int i,t;
    printf("原始数组是:\n");
    for(i=0;i<N;i++)
        printf("%d ",a[i]);
    for(i=0;i<N/2;i++)
    {
        t=a[i];
        a[i]=a[N-1-i];
        a[N-1-i]=t;
    }
    printf("\n排序后的数组:\n");
    for(i=0;i<N;i++)
        printf("%d ",a[i]);
    printf("\n");
    return 0;
}
