运算符

public class pro1 {
    public static void main(String[] args) {
        int a=3;
        int b=a++;
        System.out.println(a);
        System.out.println(b);
        int c=++a;
        System.out.println(a);
        System.out.println(b);
        System.out.println(c);
    }
}

运行结果：

总结就是：
a++是先执行完了a再加，
++a是先加上再执行
