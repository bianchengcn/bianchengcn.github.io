爱快路由系统带PPPOE服务器吗,爱快路由+华为交换机 实现vlan下拨号到爱快pppoe服务器...

本帖最后由 爱吃面条 于 2017-11-21 23:11 编辑

最近手上有一台华为二层交换机 quidway-s5700-li。然后，研究了下爱快绑定vlan拨号，隔离广播流量，有点意思。下面是自己配置，有想法的或者有其他意见的，欢迎一起交流一，爱快配置，lan口，pppoe地址池什么的基础配置就不说了

1，认证计费——认证服务管理——pppoe服务端——高级设置里，勾选绑定vlan

2，认证账号管理，添加新的账号密码时候，记得绑定vlan ID(和下面华为交换机端口 vlan ID对应起来)

二，华为二层交换机配置

其实也就是，给华为交换机划分好vlan ID，这样用户宽带拨号上网时，数据包里会带有个vlanID tag标签。然后该数据包到爱快路由器里，对照该用户绑定的vlanID，如果一致，就拨号成功，正常上网。不一致的话，就会拨不上去。下面是华为交换机vlan划分配置

1，[Quidway-s5700]vlan batch 2 to 100            创建vlan2-100

2，[Quidway-s5700]stp mode rstp                    开启stp协议

stp instance 0 priority 0

3，配置端口类型

[int g0/0/1]个人习惯，1口默认配置

[int g0/0/2]个人习惯，配制成trunk上联口，连接爱快使用

description switch  uplink port

port link-type trunk

port trunk allow-pass vlan 2 to 100

[int g0/0/3]配制成access口，并绑定vlan ID，连接客户端使用

port link-type access

port default vlan 3

[int g0/0/4]port link-type access

port default vlan 4

......

......

以此类推

[int g0/0/24]port link-type access

port default vlan 24

#return

这里是华为的二层交换机，vlan下拨号到爱快pppoe服务器。三层配置，应该还有其他方法。
