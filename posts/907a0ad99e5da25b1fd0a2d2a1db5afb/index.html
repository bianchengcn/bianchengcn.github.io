html修改文字颜色代码

要在 HTML 中修改文字的颜色，你可以使用 style 属性并设置 color 属性。

例如：

<p style="color:red">这是一段红色文字</p>

你还可以使用 16 进制颜色值或者颜色名称来指定颜色。例如：

<p style="color:#FF0000">这是一段红色文字</p>
<p style="color:orange">这是一段橙色文字</p>

你也可以在外部 CSS 样式表中定义文字颜色，然后使用 class 或 id 属性在 HTML 中应用。例如：

.red-text {
  color: red;
}

#orange-text {
  color: orange;
}

<p class="red-text">这是一段红色文字</p>
<p id="orange-text">这是一段橙色文字</p>

注意，在 HTML 中使用 style 属性通常被视为不好的编码实践，因为它把样式与内容混在一起，这会使代码难以维护和扩展。最好的做法是使用外部 CSS 样式表来统一定义样式。
