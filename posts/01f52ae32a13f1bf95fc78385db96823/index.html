Git 修改用户名

打开Gitbash

在gitbash中输入
第一步：git config user.name
查看现在的名字
第二步：git config user.email
查看邮箱
第三步：git config --global user.name “你要修改的名字”
要修改的名字写在“ ” 中
第四步：git config user.name
再次看下现在的名字，看是否修改成功
