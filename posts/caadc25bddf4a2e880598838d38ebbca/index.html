【Java练习】判断数组元素是否有序（递增）

学习目标：

目标：熟练运用 Java所学知识

题目内容：

本文内容：使用java语言实现：判断数组元素是否有序（递增）

实现思想：

想要判断数组是否递增，可以用布尔类型方法判断，如果出现的递减的相邻元素，即可判断其不是递增。

实现代码

public class practice {
    public static void main(String[] args) {


        int[] arr = {1, 2, 3, 4, 5, 6};
        System.out.println(isSort(arr));
    }
//判断相邻元素是否有递减，出现递减返回false
    private static boolean isSort(int[] arr) {
        for (int i = 0; i < arr.length-1; i++) {
            if (arr[i] > arr[i + 1]) {
                return false;
            }
        }
        return true;
    }
}

运行结果
