html设置透明度

在html中，设置元素的透明度

在css相应元素中添加下面代码即可。

opacity:0.5;

opacity的取值范围为0~1，0表示全透明，1表示不透明

完整示例代码块

div{
    opacity:0.5;
}

结合示例项目贴出代码块

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title>html透明度设置</title>
		<style>
			/* 不透明 */
			.div1{
				width: 200px;
				height: 200px;
				background-color: red;
			}
			/* 透明 */
			.div2{
				width: 200px;
				height: 200px;
				background-color: red;
				opacity: 0.5;
			}
		
		</style>
	</head>
	<body>
		<!-- 不透明 -->
		<div class="div1">
			不透明
		</div>
		<!-- 透明 -->
		<div class="div2">
			透明
		</div>
	</body>
</html>
