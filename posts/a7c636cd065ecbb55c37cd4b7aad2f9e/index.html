Syntax error: Bad for loop variable

如果会报错，没有则跳过：

Syntax error: Bad for loop variable

原因：代码对于标准bash而言没有错，因为Ubuntu为了加快开机速度，用dash代替了传统的bash，是dash在捣鬼。

解决方法：取消dash

dpkg-reconfigure dash 

出现弹框，选择NO

 

或者这样执行： bash  xxx.sh
