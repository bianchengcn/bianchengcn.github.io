使用js写一个防抖函数

使用 JavaScript 实现防抖函数的方法是：首先定义一个函数，并在其中定义一个定时器；然后在函数的末尾添加一个 if 语句，如果定时器存在，则清除定时器；最后在 if 语句外部添加一个定时器，在执行函数之后设定一个延时。这样，函数执行完毕后，延时过程中如果有新的函数调用，则取消上一次的延时，重新设定新的延时，从而实现了防抖函数。
