html怎么添加圆圈按钮,如何使用HTML5和CSS 3在圆圈周围放置按钮？

您可以使用一小部分

jquery来加载您想要的文本的中心圆.

$(document).ready(function() {

$('.quart').click(function() {

var ind = $(this).index();

switch (ind) {

case 0:

var tex = "div 1";

break;

case 1:

var tex = "div 2";

break;

case 2:

var tex = "div 3";

break;

case 3:

var tex = "div 4";

break;

}

$('.center').text(tex);

});

});

.wrap {
