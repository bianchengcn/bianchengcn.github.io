题目描述：

给你一个按 非递减顺序 排序的整数数组 nums，返回 每个数字的平方 组成的新数组，要求也按 非递减顺序 排序。

示例 1：

输入：nums = [-4,-1,0,3,10]
输出：[0,1,9,16,100]
解释：平方后，数组变为 [16,1,0,9,100]
排序后，数组变为 [0,1,9,16,100]

示例 2：

输入：nums = [-7,-3,2,3,11]
输出：[4,9,9,49,121]

解题思路：

简单一点的做法就是先将数组进行求平方，在使用Arrays.sort()进行排序

实现代码：

class Solution {
    public int[] sortedSquares(int[] nums) {
  for(int i = 0; i< nums.length; i++){
  //进行求平方
            nums[i]*= nums[i];
        }
        Arrays.sort(nums);//排序
        return nums;
        }
}
