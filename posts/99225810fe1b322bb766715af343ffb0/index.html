为什么设置请求头的过程中出现了问题？

问题如下：

Vue.http.interceptors.push((request, next) => {
	request.headers.set('lanId', 100);
})

项目运行的过程中，每个页面都会报如下错误：

后来在调试过程中才发现

设置这个方法会对字符串去除空格，而整数，没有这个方法。
有错就改掉：

Vue.http.interceptors.push((request, next) => {
    	request.headers.set('lanId', ‘100’);
   })
