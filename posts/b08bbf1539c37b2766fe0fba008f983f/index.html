代码展示

打开浏览器访问网站

driver = webdriver.Chrome()
url = 'https://www.***网址屏了*不过审*.com/kmy-mnks'
driver.get(url)

driver.implicitly_wait(10)

通过元素定位: 获取答案 / 进行答题操作

lis = driver.find_elements_by_css_selector('.Content li') 

for li in lis:
    answer = li.get_attribute('k')
    if answer == 'E':
        answer = '错误'
    elif answer == 'R':
        answer = '正确'
    bs = li.find_elements_by_css_selector('b')
    for b in bs:
        choose = b.text
        if len(choose) > 2: 
            choose = choose[0] 
        if answer == choose:
            b.click()

提交试卷

driver.find_element_by_css_selector('.btnJJ').click()
