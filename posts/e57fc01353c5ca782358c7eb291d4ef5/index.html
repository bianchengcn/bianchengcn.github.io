电脑不能调节亮度了

今天更新一下系统，电脑突然不能调节亮度。

看到这个答案受到启发：电脑突然调节不了亮度？让我教你来恢复

打开 "设备管理器" => "显示适配器" => 下面的驱动右键（两个都要） => 选择更新驱动 => 浏览我的计算机以查看驱动程序软件 => 选择 "让我从计算机上的可用驱动程序列表中选取" => 在显示兼容硬件下, 选择 "通用即插即用监视器"
