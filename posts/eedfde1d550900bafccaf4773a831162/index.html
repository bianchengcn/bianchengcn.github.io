完美国际架设mysql_完美国际SF-物理机架设服务端教程

1.上传服务端

2.上传Java

3.接下来是安装64位系统兼容32位组建 输入命令yum -y install glibc.i686

4.防火墙开启设置#/sbin/iptables -I INPUT -p tcp --dport 29000 -j ACCEPT (29000默认游戏一线端口)

#/sbin/iptables -I INPUT -p tcp --dport 39000 -j ACCEPT (39000默认游戏二线端口)

#/sbin/iptables -I INPUT -p tcp --dport 49000 -j ACCEPT (49000默认游戏三线端口)

#/sbin/iptables -I INPUT -p tcp --dport 59000 -j ACCEPT (59000默认游戏四线端口)

#/sbin/iptables -I INPUT -p tcp --dport 443 -j ACCEPT (443默认后台端口)

#/sbin/iptables -I INPUT -p tcp --dport 3306 -j ACCEPT (3306数据库端口)

#/sbin/iptables -I INPUT -p tcp --dport 88 -j ACCEPT (88jetty后台端口)

#/sbin/iptables -I INPUT -p tcp --dport 80 -j ACCEPT (80默认网页端口)

命令：/etc/rc.d/init.d/iptables save保存防火墙设置

命令：service iptables restart重启防火墙，修改生效

关闭端口命令：/sbin/iptables-I INPUT-p tcp--dport80 -j DROP

命令：/etc/rc.d/init.d/iptables save保存防火墙设置

命令：service iptables restart重启防火墙，修改生效

4.安装服务端运行坏境命令：yum-y install ld-linux.so.2

命令：yum-y install libz.so.1

命令：yum-y install libgcc_s.so.1

命令：yum-y install libstdc++.so.5

5.三.安装调试mysql

1.下载安装数据库

命令：yum-y install mysql-server

2.启动数据库

命令：service mysqld start

3.进入数据库

命令：mysql

4.更改数据库密码

命令：setpasswordfor 'root'@'localhost' =password('123456');

命令：GRANT ALL PRIVILEGES ON*.*TO'root'@'%'IDENTIFIED BY'123456'WITH GRANT OPTION;

注意：密码可随意输

5.建立数据库

命令：create database wm;

注意：数据库名可随意

6.查看数据库

命令:show databases;

7.使用数据库WM

命令：usewm;

8.上传wm.sql库文件

注意：上传到ROOT目录下

导入数据库

命令：source/root/wm.sql

9.查看表

命令：show tables;

10.退出

命令：\q

11.开机启动数据库

命令：chkconfig mysqld on

6.解压服务端

命令：ls

命令：tar zxvf123.tar.gz-C/

7.给权限

下面的命令一条一个回车命令：chmod777 /root/pwserver-R

命令：chmod777 /root/pwserver/my_authd-R

命令：chmod777 /usr/java-R

命令：chmod777 /usr/local/jakarta-tomcat-5.5.9 -R

注意：根据上传的目录给权限

8.配置AU数据库

用WINSCP打开目录到AU

路径/root/pwserver/my_authd/build/table.xml

路径/root/authd/build/table.xml

9.配置游戏线路

路径/root/pwserver/glinkdgamesys.conf

10.配置后台

路径/root/jetty/webapps/pwadmin/WEB-INF/.pwadminconf.jsp

后台启动kp 后台关闭gb

完美内侧后台路径https://192.168.1.20/iweb/role/manage.jsp

配置/usr/local/jakarta-tomcat-5.5.9/conf/tomcat-users.xml密码

基本后台路径httpss://192.168.1.20:88/pwadmin/
