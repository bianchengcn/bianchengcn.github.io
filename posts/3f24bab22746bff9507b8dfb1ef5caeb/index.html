MySQL如何获取每个分组内前n条数据

问题：

用sql实现逻辑：对一份数据tb，按照字段a, b分组，然后每个分组按照字段x, y升序排序，每个分组取前n条数据。

解决方案：

select * 
from (
	select 
	dt.*, 
	@_rn := if(@_prev_a = a and @_prev_b = b, @_rn + 1, 1) as _rn,
	@_prev_a := dt.a as _prev_a,
	@_prev_b := dt.b as _prev_b
	from tb dt join (select @_rn:=0, @_prev_a:=null, @_prev_b:=null) tmp
	order by dt.a, dt.b, dt.x, dt.y
) where rn <= n

解释

  1. 首先用分组字段+排序字段进行排序（order by dt.a, dt.b, dt.x, dt.y）
  2. 初始化所有临时变量（select @_rn:=0, @_prev_a:=null, @_prev_b:=null）
  3. 利用临时变量@_prev_a和@_prev_b来记录上一行分组字段的值
  4. 用@_rn在每一个分组中递增+1，当到达新分组的时候置为1（@_rn := if(@_prev_a = a and @_prev_b = b, @_rn + 1, 1)）

注：@variable在mysql里代表临时变量，@variable:= value用来对临时变量进行赋值，变量在运算中会逐行按照当前值进行操作
