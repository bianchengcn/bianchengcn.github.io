通过指针消除字符串重复项

思路：定义两个指针，一个指针p在前固定一个值，一个指针q在后从后往前与*p的值相比较，若相同则把后一项值赋值给前一项。

#include<stdio.h>
#include<string.h>
int main(int argc, const char *argv[])
{
	char s[32]={0},i,j;
	char *p = s;
	char *q = s;
	printf("请输入字符串：");
	gets(s);
	for(i=0;i<strlen(s);i++)
	{
		for(j=strlen(s);j>i;j--)
		{
			if(*(p+i)==*(q+j))
			{
				*(q+j)=*(q+j+1);
			}
		}
	}
	printf("去重结果为：%s\n",p);
	return 0;
}
