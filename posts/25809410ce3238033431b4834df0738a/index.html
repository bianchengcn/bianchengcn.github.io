vue调用打印机（vue-print-nb）

通过npm 安装插件

1，安装 npm install vue-print-nb --save

2，引入 安装好以后在main.js文件中引入

 import Print from 'vue-print-nb'

Vue.use(Print); //注册

3，现在就可以使用了

<div id="printTest" >

　　　　　　<p>锄禾日当午</p>

　　　　　　<p>汗滴禾下土 </p>

　　　　　　<p>谁知盘中餐</p>

　　　　　　<p>粒粒皆辛苦</p>

　　　　</div>

　　　　<button v-print="'#printTest'">打印</button>

如果内容打印不全，在打印操作时点击更多设置，然后设置缩放

常见问题

  1. 横向打印明明一页就能打印完，但是却会打印两页， 请检查打印区域顶部或者底部是不是有溢出的magin。
  2. 打印区域的样式错乱了， 请检查css，尽可能的将选择器写完整一点，如：#printTest .xxx 。
  3. 打印出来的结果紧贴在A4纸边缘，这个只需要给打印区域一个padding内边距即可。
