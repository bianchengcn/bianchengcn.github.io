更新node之后重新安装node_modules报错

更新node版本之后一定要重新安装依赖。

安装依赖的时候可能会遇到各种各样的报错，具体解决方法如下：

1.全局安装npm包：

npm install rimraf -g

2.删除 node_modules 文件夹命令：

rimraf node_modules

3.清理npm缓存：

npm cache clean --force

4.重新安装依赖库：

npm install

以上方法能解决90%的node更新之后安装依赖时的报错，over~
